# Expense Tracker Web App

A modern Flask-based Expense Tracker Web Application with advanced analytics, beautiful UI, secure authentication, admin panel, and smart expense management.

---

## Features

* User Authentication System
* OTP Verification
* Two-Step Verification
* Secure Password Hashing
* Expense Tracking
* Monthly Budget Management
* Expense Categories
* Expense Image Upload
* Expense Gallery
* Analytics Dashboard
* Income Tracking
* Login Activity Monitoring
* Admin Panel
* Notification Settings
* Profile Picture Upload
* CSV / Excel Export
* Beautiful Responsive UI
* Dark Futuristic Design

---

## Tech Stack

* Python
* Flask
* SQLAlchemy
* Bootstrap 5
* HTML/CSS
* JavaScript
* SQLite

## Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/ExpenseTrackerWebApp.git
```

Go into project folder:

```bash
cd ExpenseTrackerWebApp
```

Create virtual environment:

```bash
python -m venv venv
```

Activate virtual environment:

### Windows

```bash
venv\Scripts\activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```
